# Getting started

install bigbud on your machine, launch the app, and start your first session. no terminal needed — everything works from the app.

---

## 1. Download and install

bigbud runs on macOS, Windows, and Linux.

**[Download the latest version](/download)** — the installer will walk you through setup in a few clicks.

---

## 2. Launch the app

open bigbud from your applications folder (or start menu on Windows). you'll see your workspace — a clean window with a sidebar on the left and the main area on the right.

---

## 3. Connect an AI provider

bigbud works with the AI providers you already use. to get started:

1. open **settings** (click the gear icon in the sidebar or press `mod+,`)
2. go to the **providers** section
3. choose a provider and sign in or paste your API key

supported providers: **Claude**, **Codex**, **Copilot**, and **OpenCode**. you can connect more than one and switch between them as you work.

once connected, you're ready to start.

---

## 4. Start a thread

press `mod+n` to create a new thread. type your question or task in the input at the bottom and press enter. bigbud will respond using the active provider.

threads are organized in the sidebar — you can switch between them, rename them, or archive them as you go.

---

## 5. Explore your workspace

bigbud brings everything into one window:

- **chat** — talk to your AI providers, ask questions, get help
- **terminal** — press `mod+j` to open a terminal inside the workspace
- **files** — press `mod+shift+e` to browse and open files from your projects
- **diff bar** — press `mod+shift+g` to review changes side by side
- **search** — press `mod+f` to search across your workspace

you can resize or hide panels to suit your flow.

---

## 6. Keyboard shortcuts

| shortcut                      | action                 |
| ----------------------------- | ---------------------- |
| `mod+j`                       | toggle terminal        |
| `mod+b`                       | toggle sidebar         |
| `mod+,`                       | open settings          |
| `mod+f`                       | search workspace       |
| `mod+shift+g`                 | toggle diff bar        |
| `mod+shift+b`                 | toggle browser panel   |
| `mod+shift+e`                 | toggle files panel     |
| `mod+p`                       | command palette        |
| `mod+n`                       | new thread             |
| `mod+shift+n`                 | new local thread       |
| `mod+shift+[` / `mod+shift+]` | previous / next thread |

(`mod` is `Cmd` on macOS and `Ctrl` on Windows/Linux)
